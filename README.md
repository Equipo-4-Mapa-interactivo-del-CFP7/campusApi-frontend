# Base CFP7

Este es el proyecto (base)Frontend de la AppWeb para CFP7. Es la interfaz visual que permitiría complementar la gestión y navegación.

Este proyecto está construido con [Next.js](https://nextjs.org/)

## Requisitos Previos

Antes de empezar, asegúrate de tener instalado en tu computadora:
- [Node.js](https://nodejs.org/) (necesario para gestionar los paquetes e instalar dependencias).

## Instalación y Uso

Pasos para levantar el proyecto en tu entorno local:

1. **Clonar el repositorio:**
   Ubícate en la rama de desarrollo (`dev`) para obtener los últimos cambios:
   ```bash
   git clone -b dev https://github.com/hollywoodOK/BaseFrontCFP7h.git
